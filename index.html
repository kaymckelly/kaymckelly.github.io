About

How I Got Here

I've been a bonafide geek since 1998, when I spent my free time building websites on Geocities, editing CSS for my Livejournal layouts, and writing my class notes in HTML. In 2015 I started teaching myself jQuery to see if coding was really where I wanted to be. Seven months later, I moved across the country to start at Galvanize Seattle and have never looked back.

After graduating from Galvanize's 6 month software development program in July 2016, I've been working as a freelance web developer for the International Association for Cryptologic Research (IACR) and Farcaster Consulting Group.

Last April, I volunteered to teach an Introduction to Angular class with She Codes Now.
