Portfolio

Boost

I built this Capstone project in two weeks. With Angular 1 on the front-end, an Express server and PostgreSQL database on the back-end, this app connects strangers to help each other out. Boost also teaches good self-care techniques. It includes a meditation timer, inspirational quotes, and a self-care checklist.

See the code on Github

SNC

A group project built in a week, Seattle Neighborhood Challenge is a React Native app with an Express server. We chose React Native so SNC could function as a native app on both ios and android. I acted as project manager, in addition to contributing to the codebase. We used the Google Maps API, oAuth with Facebook login, Knex query builder, and an Agile workflow.

See the code on Github

Brows on Fleek

Brows on Fleek was the first project I completed while at Galvanize. Designed to help people to find places to get their eyebrows done within a given radius, this project was focused on front-end development and using APIs. It's a front-end project that utilizes Twitter Bootstrap, HTML5, CSS3, and the Google Maps API.
