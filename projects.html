Introduction

This is a visual showcase of my favorite projects. Whether it's design or topic (or, more often, both), I really enjoyed building these projects. I strongly believe that you should love what you do, so when I take on a project, it's going to be something I enjoy. There's always more work to do, so this page may not reflect updates to these apps. Please see Github for the code and latest refactoring.

I do my best to adhere to web accessibility standards as set forth by World Wide Web Consortium (W3C). This particularly affects color contrast, in order to make apps accessibile to users with impaired vision. Gray-on-gray low-contrast text is in fashion right now and while aesthetically I agree it looks nice, I'm more invested in making apps and websites that are truly accessible for all users.

Freelance Work for IACR

The International Association for Cryptologic Research (IACR) requested a conference website redesign. IACR was primarily interested in making their conference site mobile-accessible, easily maintainable, and have a more modern look. They wanted a site they could re-use for other conferences they host, so that all the conference pages would have a standardized look.

Their last conference site (shown above at left) resembled sites built in the late 1990s/early 2000s and was not mobile accessible. Additionally, the site was almost exclusively written using HTML 1.0, which debuted in 1991.

My redesign (shown at right) is more colorful, with responsive menus and a mobile friendly design using Bootstrap 3.3. I created the color palette for the site based on IACR's logo, with bright saturated shades of orange and green that complement the deep indigo of their logo. This project is still in progress, but there is a live preview available here.
